Como é possível ver, existem 2 arquivos com o mesmo nome, um arquivo .txt e um .ino
Foi feito isso para uma ajuda ao usuário, pois o arquivo txt lhe permite entrar no código sem problema, para uma rápida visualização.
Caso haja interesse em testar o código, fazer modificações e mais dentro do TinkerCad, recomendo abrir o arquivo .txt para somente copiar e colar o código
dentro do próprio TinkerCad, e de lá fazer suas alterações e aprender com experiência.

Já o arquivo .ino foi feito para aquela pessoa que possui interesse em testar o código na sua própria microcontroladora (Fora do tinkercad), já que o arquivo .ino
 é o formato utilizado para a compilação dentro de Arduinos e outras placas similares.

A lingaguem utilizada durante o TinkerCad e para programação de Arduino é a C++, com pequenas mudanças, como a adição do setup( ) e substituição do 
main( ) pelo loop( )
